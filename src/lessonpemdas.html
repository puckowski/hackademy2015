Intel Report 1: What are the Order of Operations?

If you are asked to simplify a problem such as “ 5 + 4 x 3”, how would you go about solving this?
There are many ways to go about solving this, most people might solve it from left to right just like when you read from left to right. But that approach is incorrect. The correct way to solve this is to use the order of operations (PEMDAS). If you can’t remember PEMDAS, an easy way to remember the order is shown below:

"Please Excuse My Dear Aunt Sally"

1. Parentheses
2. Exponents
3. Multiplication
4. Division
5. Addition
6. Subtraction

This acronym will helps to explain the order of operations. First you will compute what is inside the parentheses. Then exponents come next, then multiplication and division, and then addition and subtraction. This is generally the order but in some cases if division comes before multiplication you compute that first and the same goes for subtraction before addition.

Intel Report 2: Examples

Example 1.
(4 x 3) + 5
First, compute what is inside the parentheses because it has the highest status.
4 x 3 = 12
Then you compute the rest of the equation which is, 12 + 5 = 17.
The final answer is 17.

Example 2.
(20 ÷ 4) – 32
This simplifies to: 5 - 9 = -4.

Example 3.
52 x 6 – 40 + 3 =
First compute the exponent part, 52 = 25.
25 x 6 is the next step, the product is 150.
Now, we compute 150 - 40 = 110 and add 3. The final answer is 113.
